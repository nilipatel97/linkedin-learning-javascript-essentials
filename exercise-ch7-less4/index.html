Build and Modify an Array

Instructions:

  1. Create an array with several (5-8) items
  2. Remove the last item
  3. Move last item to first position
  4. Sort items alphabetically
  5. Find specific item in the array
  6. Remove an item with specific content from array (find & remove)

Tips:

  • Remember, when working with arrays, you can nest methods inside other methods.
  • Also, can explicitly create variables and put array into the variable, then outputting later.
  • MDN Web docs shows how to use multiple methods together.
  • Main purpose of exercise: use MDN web docs to develop a document-driven practice
