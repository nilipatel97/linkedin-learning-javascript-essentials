Chapter 5 Lesson 10 Challenge: Add a New Element

High-level Overview

  1. make a nav menu right under the header
  2. write the javascript and add basic css
  3. horizontal menu

Steps

  1. Create a new element to hold menu
  2. Add an unordered list of at least 5 links (src=#)
  3. Add new navigation menu to the DOM directly after the header
  4. Write basic CSS classes to create horizontal menu. TIP: use display flex or display grid
